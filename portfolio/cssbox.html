Box Models And CSS Definitions

Cascading Style Sheets (CSS) - Cascading Style Sheets is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML.

CSS cascade - the cascade is an algorithm that defines how user agents combine property values originating from different sources.

CSS inheritance - describes how certain styles are initialized or computed depending on the CSS property and whether a value was set.

CSS source order - sets the order to lay out an item in a flex or grid container.

CSS specificity - the algorithm used by browsers to determine the CSS declaration that is the most relevant to an element, which in turn, determines the property value to apply to the element.

CSS rule - contains one or more selectors and one or more declarations.

CSS declaration - the key-value pair of a CSS property and its value.

CSS property - determines an HTML element's style or behavior.

CSS value - used to define the set of valid values for CSS property of function.

CSS selector - used to target the HTML elements on our web pages that we want to style.

id selector - uses the id attribute of an HTML element to select a specific element.

class selector - selects elements with a specific class attribute.

child selector - selects all elements that are the children of a specified element.

sibling selector - used to select elements that are siblings (share the same parent element)

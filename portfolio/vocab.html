Programming

vocabulary

Layer

a distinct level within a hierarchical structure that separates different functions or responsibilities within a system.

synonyms

  1. Service
  2. Tier

User Interface

the point of contact between humans and computers.

synonyms

  1. Front end
  2. UI

Software

the programs and other operating information used by a computer.

synonyms

  1. Program
  2. Script

Application

computer program designed to help people perform an activity.

synonyms

  1. program
  2. software

user-interface designer

A UI designer is someone who designs the graphical user interface of an app, website, or device that a human interacts with.

synonyms

  1. Front end

front-end developer

Front-end development focuses on the user-facing side of a website.

synonyms

  1. Web designer
  2. CSS/HTML developer

back-end developer

Backend developers build code that allows a database and an application to communicate with one another.

synonyms

  1. database developer
  2. database

database developer

responsible for the design, programming, construction, and implementation of new databases, as well as modifying existing databases for platform updates and changes in user needs.

synonyms

  1. database designers
  2. database programmers

pattern

a relatively small, well-defined aspect (i.e. functionality) of a computer program in terms of how to write the code.

ui layer

The User Interface (UI) layer sits above the Logic layer, the user can interact with it.

synonyms

  1. front end

abstraction-abstract

abstraction is the process of generalizing concrete details, such as attributes, away from the study of objects and systems to focus attention on details of greater importance.

logic layer

the part of a program where classes determine the information needed to solve problems, manipulate that information, and return results to the presentation layer for formatting.

persistence layer

Any software layer that makes it easier for a program to persist its state

separation of concerns

a design principle for separating a computer program into distinct sections.

stack

The stack is the 3 layers ( Front end, Logic, Back End)

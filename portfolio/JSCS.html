JavaScript Cheat Sheet

let Re-assignable Variable use this if you want to change a variable throughout the code.
const Variable that cannot be re-assigned use this if you dont want a variable to change throughout the code.
for loop Use this if you want to run the same code a fixed amount of time.
Array.forEach() Calls a function for each element with the same variable use this if you want to call a function with the same variable ( const, let )
document.getElementById() The getElementById() method returns an element by its Id. use this if you want to get an element by its id
document.getElementsByTagName() Method returns a live HTML collection of an element with the given tag name. use this if you want to get an element by its name
document.getElementsByClassName() method of Document interface returns an array-like object of all child elements which have all of the given class name(s). use this if you want to get an element by its class name
document.querySelector() The querySelector() method returns the first element that matches a CSS selector.
document.querySelectorAll() Query selector is a method in JavaScript that is used to grab classes, ids and tags easier. Use this if you want to grab a bunch of classes, ids and tags easier.

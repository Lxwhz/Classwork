Id & Class Selectors

The id selector uses the id attribute to select a specific HTML element, The id of an element is unique so the id selector is used to select one unique element.

The class selector uses the class attribute to select specific HTML elements.

Examples of ID

  1. #EX1 {}
  2. #EX21 {}
  3. #EX321 {}

    4. To select an element with a specific id, write a hashtag (#) character, followed by the id of the element.

Examples of Class

  1. .EX1 {}
  2. .EX21 {}
  3. .EX321 {}

    4. To select elements with a specific class, write a period (.) character, followed by the class name.
